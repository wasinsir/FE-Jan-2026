# Products Management System

Products Management is a web application for managing product data for SME in Thailand 

This project built on [Vite](https://vite.dev/), [React](https://react.dev/), [TypeScript](https://www.typescriptlang.org/)

# Features
- **Table of Products**
  - The user can filter products by category.
  - User can add a new product to the system to track the stock of products 
  - Allow the user to record sales data by using the sell button.
  - At the bottom of the table, the system shows the total value of all products
  
# Getting Started
**Prerequisites**

- [Node.js](https://nodejs.org/en) (version 18.x or higher is recommended) 
- Package manager (NPM)

# Run project
Install dependencies:
   ```sh
   npm install
   ```

Run
   ```sh
   npm run dev
   ```


# Library
- Routing: [React Router](https://reactrouter.com/)
- State management: [Redux](https://redux-toolkit.js.org/)
- Testing: [Vitest](https://vitest.dev/) & [React Testing Library](https://testing-library.com/)
- Component: [styled-component](https://styled-components.com/)
